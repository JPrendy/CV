A portfolio website I built using tested and documented HTML, CSS, jQuery and JavaScript. Built with Web Frameworks such as AngularJS and Bootstrap 4 to allow for responsive and user-friendly design.

This is a website I constructed as a side project, to utilise the knowledge I gained from my years of studying web design. This is the second version of my portfolio website. It uses frameworks and techniques to improve performance and provide a much better layout to make changes quicker.

It displays a digital version of my Curriculum Vitae and demonstrates some projects I created while during my studies of Computer Science. Included with each project is a description, some images, a video of the project in action and the Git Repository.

I am happy to see this website live. There is not a better feeling to see something you put hardwork into be available online.
